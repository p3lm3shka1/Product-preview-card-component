## Responsive design made with a little bit of SASS :) 

```css
@media (max-width: 375px) {
  body {
    height: auto;
    padding: 20px 0;
  }

  .container {
    flex-direction: column;
    width: 90%;
    height: auto;
    border-radius: 0.7rem;
    overflow: hidden;
  }

  .left-side {
    img {
      content: url("images/image-product-mobile.jpg");
      width: 100%;
      height: auto;
    }
  }

  .right-side {
    padding-bottom: 20px;
  }
}
```
